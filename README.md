# Challenge1_AluraStore
## **El propósito del análisis realizado**

Este análsiis tiene como propósito y objetivo principal mostrarle al Sr. Juan, cual de sus tiendas es la menos favorable para mantener en operación y por lo tanto, la mejor opción para vender, debido a los bajos resultados que presenta según sus datos. Todo esto sustentando por una serie de análisis hechos en base a los datos proporcionados.
## **La estructura del proyecto y organización de los archivos.**
El presente proyecto se realizó en un notebook de Google Collab, el cual, se estructuró de la siguiente forma: 

### 1. Importación de datos
### 2. Análisis de facturación
### 3. Ventas por categoría
### 4. Calificación promedio de la tienda
### 5. Productos más y menos vendidos
### 6. Envío promedio por tienda
### 7. Visualizacion
### 8. Gráficos
### 9. Informe de rendimiento de las tiendas

Como se puede observar, para crear un notebook claro y organizado, dividimos el notebook en secciones para ir completando cada campo de una forma organizada. Todo esto con el fin de lograr un análisis limpio, organizado y claro de los datos de las tiendas del Sr. Juan.
